# محافظي - تطبيق إدارة المحافظ الإلكترونية

تطبيق SaaS متكامل لإدارة المحافظ الإلكترونية، مصمم بنظام Mobile-First لمساعدتك على تتبع أموالك وإدارة معاملاتك المالية بسهولة.

## الميزات الرئيسية

### 1. **إدارة المحافظ الإلكترونية**
- إنشاء محافظ متعددة (عامة، توفير، استثمار، ائتمان)
- تخصيص الألوان والأيقونات لكل محفظة
- دعم متعدد العملات (ريال سعودي، دولار أمريكي، يورو، إلخ)
- عرض الرصيد الحالي لكل محفظة

### 2. **تتبع المعاملات**
- إضافة معاملات دخل ومصروف
- تصنيف المعاملات حسب النوع
- تخصيص التصنيفات للمعاملات
- تسجيل التواريخ والملاحظات
- حالة المعاملة (مكتملة، معلقة، ملغاة)

### 3. **التصنيفات المالية**
- تصنيفات جاهزة للدخل (راتب، مكافأة، استثمار، هدايا، إلخ)
- تصنيفات جاهزة للمصروفات (طعام، نقل، فواتير، تسوق، إلخ)
- إمكانية تخصيص الألوان والأيقونات

### 4. **لوحة المعلومات**
- عرض إجمالي الرصيد من جميع المحافظ
- إجمالي الدخل والمصروفات
- آخر المعاملات
- نظرة شاملة على جميع المحافظ

### 5. **التقارير والإحصائيات**
- تصفية المعاملات حسب المحفظة
- عرض سجل المعاملات الكامل
- إحصائيات سريعة لكل محفظة

## التقنيات المستخدمة

### Frontend
- **Next.js 15** مع App Router
- **TypeScript 5**
- **Tailwind CSS 4**
- **shadcn/ui** للمكونات الجاهزة
- **React 19**
- **Lucide React** للأيقونات

### Backend
- **Next.js API Routes**
- **Prisma ORM** مع SQLite
- **TypeScript**

### قاعدة البيانات
- **SQLite** للتخزين المحلي
- **Prisma** كـ ORM

## هيكل قاعدة البيانات

### Wallet (المحفظة)
- معلومات المحفظة (الاسم، الوصف، الرصيد)
- نوع المحفظة (عامة، توفير، استثمار، ائتمان)
- العملة والألوان والأيقونات
- العلاقة مع المستخدم والمعاملات

### Transaction (المعاملة)
- تفاصيل المعاملة (العنوان، الوصف، المبلغ)
- نوع المعاملة (دخل، مصروف)
- التاريخ والحالة
- العلاقة مع المحفظة والتصنيف

### Category (التصنيف)
- اسم التصنيف ووصفه
- نوع التصنيف (دخل، مصروف)
- الأيقونة واللون
- العلاقة مع المعاملات

### User (المستخدم)
- معلومات المستخدم الأساسية
- العلاقة مع المحافظ

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ أو Bun
- Git

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd wallet-management-app
```

2. **تثبيت الحزم**
```bash
bun install
# أو
npm install
```

3. **إعداد قاعدة البيانات**
```bash
bun run db:push
```

4. **تهيئة البيانات الأولية**
```bash
bun run db:seed
```

5. **تشغيل التطبيق**
```bash
bun run dev
```

6. **فتح المتصفح**
افتح [http://localhost:3000](http://localhost:3000)

## الأوامر المتاحة

```bash
# تشغيل خادم التطوير
bun run dev

# بناء التطبيق للإنتاج
bun run build

# تشغيل خادم الإنتاج
bun run start

# فحص الكود
bun run lint

# تحديث قاعدة البيانات
bun run db:push

# إنشاء Prisma Client
bun run db:generate

# تهيئة البيانات الأولية
bun run db:seed
```

## واجهة المستخدم

### الصفحة الرئيسية - نظرة عامة
- عرض بطاقات ملخصة (إجمالي الرصيد، الدخل، المصروفات)
- قائمة بجميع المحافظ مع أرصدتها
- آخر 10 معاملات

### صفحة المحافظ
- عرض تفصيلي لجميع المحافظ
- معلومات كاملة لكل محفظة
- عدد المعاملات في كل محفظة

### صفحة المعاملات
- سجل كامل لجميع المعاملات
- تصفية حسب المحفظة
- تفاصيل كل معاملة

## تصميم Mobile-First

التطبيق مصمم ليعمل بشكل ممتاز على جميع الأجهزة، مع التركيز على:

- تخطيط متجاوب يتكيف مع جميع أحجام الشاشات
